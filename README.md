# AI Service Business — Co-Founder Workspace

Working repo for building a $50k–$100k/yr AI service business that evolves into SaaS.

**Strategy: Service → Automation → SaaS.** We sell outcomes to real businesses, do the work
manually first, automate after ~5 clients, and productize only workflows customers already pay for.

## Structure

| File | Purpose |
|---|---|
| [`SCOREBOARD.md`](SCOREBOARD.md) | Running scoreboard of every idea we evaluate |
| [`opportunities/`](opportunities/) | One deep-dive per idea (framework: problem → manual service → automation → SaaS) |
| [`sales/`](sales/) | Cold emails, scripts, pricing, objection handling per offer |

## Operating rules

1. No code until 2 paying (or committed pilot) customers for the workflow.
2. Every idea gets scored in `SCOREBOARD.md` before we spend more than a day on it.
3. Sell outcomes, never technology. ("Every lead followed up in 60 seconds", not "AI + Twilio".)
4. Before building anything, answer: **how could we sell this manually first?**
